CryoChaos Screen Transform Patch

Copy these files into your existing CryoChaos project, preserving their folders,
and choose Replace when prompted.

The patch adds:
- Upside Down
- Sideways Left
- Sideways Right
- Mirror Mode
- Vertical Flip
- Windows Magnification API live capture
- x64 build target
- Screen-transform service integration
- Correct parsing for Destiny's "extra mouse button 1/2" binding names

Recommended Destiny display mode: Borderless Windowed.
Build target: x64.
